# UI Design Skill: Modern Technical Editorial

## Overview
This skill defines the user's preferred **Modern Technical Editorial** aesthetic. It blends the warmth of editorial design (serifs, cream bg) with the precision of technical/space design (lines, wireframes, mono fonts).

## Core Rules

1.  **BACKGROUND**: Always `#F9F7F2` (Cream). Never pure white.
2.  **TYPOGRAPHY TRIAD**: 
    *   **Serif**: Headings (`Playfair`, `Young Serif`)
    *   **Sans**: Body (`DM Sans`)
    *   **Mono**: Labels/Nav/Data (`JetBrains Mono`)
3.  **TECHNICAL LINES**: Use subtle 1px borders (`#2A1B12/10`) to create structure and grids.
4.  **ACCENT**: Use Vibrant Orange (`#FF6B4A`) for key interactions.
5.  **LAYOUT**: Open, spacious, with occasional "Massive" typography.
