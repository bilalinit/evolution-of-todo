# Design Tokens: Modern Technical Editorial

A sophisticated blend of **Warm Editorial** (trust, elegance) and **Technical Space** (precision, innovation).
Characterized by cream backgrounds, strong serif headings, sharp technical lines, and vibrant orange accents.

## üé® Color Palette

### Base Colors
- **Background**: `#F9F7F2` (Warm Cream) - Main page background
- **Surface**: `#F0EBE0` (Darker Cream) - Cards, sections
- **Structure**: `#E5E0D6` (Wireframe Grey) - Borders, dividers, technical lines
- **Text Primary**: `#2A1B12` (Espresso) - Headings, main text
- **Text Secondary**: `#5C4D45` (Mocha) - Supporting text

### Accents (The "Technical" Pop)
- **Primary Accent**: `#FF6B4A` (Vibrant Orange) - Primary buttons, highlights, active states
- **Secondary Accent**: `#2A2A2A` (Charcoal) - Dark mode sections, footers

### Usage Rules
- **Technical Lines**: Use 1px borders in `#E5E0D6` or `#2A1B12/10` to create structure (see Sciemo/Moonlet refs).
- **Orange Power**: Use `#FF6B4A` sparingly for high-impact calls to action (Buttons, "Book Demo").

**Font**: `JetBrains Mono`, `IBM Plex Mono`, `Space Mono`.
**Usage**: Labels, overlines, data points, navigation links.
- **Label**: 0.75rem (12px) - Uppercase, tracking wider (0.05em)
